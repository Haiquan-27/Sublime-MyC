# Sublime-MyC
简单的C语言编译工具

##开始:  My C 菜单栏 

              -> 新建

##编译:  选择 `*.c` 文件  

			-> 右键/标签
               -> `My C`
               -> 新建      `[ctrl+g]`
               -> 编译      `[f9]`
               -> 运行      `[f10]`
               -> 编译运行  `[shift f10]`    

##更改文件编码: 右键/标签 

               -> `My C`
               -> 重置文件编码为UTF-8
               -> 重置文件编码为GBK
